# Anki Pan

絵と音声と隠し文字で学ぶ、暗記アプリ。 https://anki-pan.now.sh/

## Infrastructure

* Auth0 - https://auth0.com/jp/
* Vercel - https://vercel.com/
* FaunaDB - https://fauna.com/

##  Setup

```shell
npm i
```

## Mock (with am-mocktimes)

```shell
npm start
```

## Storybook

```shell
npm run storybook
```

## Unit test

```shell
npm test
```
